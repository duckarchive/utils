# @duckarchive/utils

Utility functions for duckarchive projects, including parsing and formatting functions for database parameters, codes, and titles.

## Installation

### From GitHub (recommended)

Using pnpm:

```bash
pnpm add github:duckarchive/utils
```

Using npm:

```bash
npm install github:duckarchive/utils
```

Using yarn:

```bash
yarn add github:duckarchive/utils
```

Or add directly to `package.json`:

```json
{
  "dependencies": {
    "@duckarchive/utils": "github:duckarchive/utils"
  }
}
```
